---

## 手順

1. ユーザーがいくつかのPRのリンクを貼り付けます。

2. それぞれのPRについて、Skillツールで `export-pr-review` スキルを**必ず** を呼び出してまとめて内容を取得し、以下の観点からコメントを重要度順に選び、コメントへのリンク、引用文とともにまとめ、チャットに出力してください。出力する前にフォーマットにしたがっているかを

観点
@CLAUDE.md を参照

- コードレビューコメントとして重要なもの
- 他の解法や学びになるtipsなど
- 面白いこと・言語の内部実装に関連することなど
- Computer Scienceに関連すること
- フォローアップなど

形式

```md
### 内容のサマリ

[PRの作成者のGitHub ID#PR number](該当PRまたはコメントのリンク) // このフォーマットを逸脱しない

> 引用
> まとめ、解説など短く

### 内容のサマリ

...
```

注意:

- PRの作成者はPRのリンクからユーザー名を抽出してそれを用いてください
- 上記のフォーマットに厳密に従い、コメンテーターのユーザー名など余計な情報は**絶対に**載せないでください

3. 最後に過去に解いた類題のリンクを箇条書きで書いてください。過去解いた類題は本リポジトリに格納してあります
